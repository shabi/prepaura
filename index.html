FAQ

Common questions

Is it really invisible to screen sharing?
Yes. PrepAura uses the setContentProtection(true) and skipTaskbar OS flags, which cause the window to be excluded from any screen recording or sharing session at the OS level.
Does it cost anything to use after buying?
No. PrepAura uses Groq's API which has a generous free tier. You sign up for a free Groq account, get an API key, and paste it in the app. No extra subscription needed.
Can I use it for coding interviews on HackerRank or CoderPad?
Yes. Use the "Capture Screen" button (⌘⇧C) to screenshot the coding problem. PrepAura will analyze it and give you a solution with explanation — all hidden from the browser tab being shared.
What roles and languages does it support?
Any software engineering role. Set your role and tech stack in Settings (e.g. "iOS Engineer — Swift, UIKit, SwiftUI") and answers are tailored to your stack. Supports Swift, Python, JavaScript, TypeScript, Java, C++, Kotlin, Go, Rust, C#.
Does it work on Windows?
Yes, Windows support is included in the same license. Download the Windows installer from your purchase link.
Can I share my license key with others?
No. Each license key is for one user. Keys are single-use. If you share your key, it violates the terms of purchase.
